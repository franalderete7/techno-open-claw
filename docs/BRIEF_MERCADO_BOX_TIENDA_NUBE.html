Brief de marca y assets para Tienda Nube

Mercado Box

Documento de trabajo para una primera iteracion creativa enfocada en identidad visual y assets concretos para la home de Tienda Nube.

Fecha: 20 de abril de 2026

Preparado por: Aldegol

Marca: Mercado Box

Tienda actual: https://mercadoboxx.mitiendanube.com

Plataforma: Tienda Nube

Diseno actual identificado: Toluca

Referencia visual: https://www.mercadotop.com.ar

1. Objetivo

Quiero que este brief sirva para una primera iteracion de diseno enfocada no solo en el logo o la identidad general, sino tambien en las piezas visuales concretas que hoy necesita mi Tienda Nube.

En esta etapa, la prioridad no es redes sociales.

La prioridad es construir:

  • una identidad de marca propia para Mercado Box
  • un icono o isotipo fuerte
  • un sistema visual claro
  • y los assets especificos que necesita la home de mi tienda

2. Contexto general

Mercado Box es una marca de e-commerce que quiero hacer crecer fuerte en Argentina.

Mensaje principal: envios gratis a todo el pais, seguros y en pocos dias.

Quiero que esa promesa se sienta en la identidad visual, en el icono y en los banners principales de la tienda.

La marca deberia transmitir:

  • confianza
  • cercania
  • practicidad
  • orden
  • rapidez
  • experiencia de compra simple

3. Direccion creativa

La idea central de la marca sigue siendo una caja.

Me interesa especialmente explorar una caja que se sienta:

  • segura
  • agradable
  • moderna
  • funcional
  • con una leve sensacion de movimiento

Mi preferencia actual es una caja simple, simpatica y confiable, mas iconica que caricaturesca.

4. Inspiracion y limite de la inspiracion

La referencia que me gusta hoy es Mercado Top: https://www.mercadotop.com.ar

Lo que me atrae:

  • energia comercial
  • aspecto agradable
  • sensacion de tienda activa
  • lenguaje visual vendedor

Lo que no quiero:

  • no quiero copiarla
  • no quiero reutilizar recursos ajenos
  • no quiero que Mercado Box parezca una marca derivada

5. Situacion actual de la tienda

Hoy la tienda necesita una limpieza visual y un reemplazo completo de piezas temporales o heredadas.

  • el carrusel principal tiene contenido visual tomado de Mercado Top y debe reemplazarse por piezas 100% originales de Mercado Box
  • varias secciones de la home todavia estan vacias
  • algunos links y recursos actuales todavia remiten a contenido anterior y deben quedar completamente reemplazados

6. Prioridad de trabajo

  1. identidad visual base de Mercado Box
  2. assets de home para Tienda Nube
  3. coherencia entre logo, icono, banners y mensajes principales

No hace falta enfocarse ahora en piezas para redes sociales.

7. Entregables prioritarios

A. Logo e identidad base

  • logo principal
  • version horizontal
  • version compacta
  • isotipo o icono
  • favicon
  • paleta cromatica base
  • tipografia sugerida

Tamanos utiles para Tienda Nube: logo `575 x 230 px`, favicon `130 x 130 px`.

B. Carrusel principal

Quiero una primera iteracion con 3 banners principales para el carrusel.

Tamanos verificados para Toluca:

  • desktop: `1580 x 700 px`
  • mobile: `820 x 1000 px`

Pedido:

  • disenar version desktop y mobile por separado
  • que no sea una simple adaptacion automatica
  • que el contenido este pensado para verse bien en cada dispositivo
  • cada slide deberia tener una idea clara, limpia y rapida de entender

Lineas sugeridas para los 3 slides:

  • Envios gratis a todo el pais
  • Compra segura y en pocos dias
  • Tecnologia, ofertas y experiencia simple

C. Banners de categorias

Me interesa ver 3 o 4 banners iniciales para categorias principales.

Tamano oficial encontrado para Toluca y su familia: `930 x 465 px`.

Posibles categorias iniciales:

  • Celulares
  • Audio
  • Accesorios
  • Ofertas

D. Banners promocionales

Quiero una primera entrega de 2 o 3 banners promocionales.

Base tecnica recomendada: composicion horizontal tipo `930 x 465 px`.

Tienda Nube no publica una tabla igual de explicita para el banner promocional de Toluca, pero la estructura real del tema usa una proporcion horizontal `2:1`, consistente con el sistema de banners de esta familia. Quiero trabajarlos con esa base.

Ideas posibles:

  • envio gratis a todo el pais
  • descuentos u ofertas especiales
  • cuotas o compra segura
  • lanzamientos destacados

E. Banners informativos

Quiero resolver 3 banners informativos.

Tamano visible en el editor actual: imagen o icono `120 x 120 px`.

Ademas de la imagen, cada banner lleva:

  • icono tematico
  • titulo
  • descripcion breve
  • link opcional

Direccion de contenido sugerida:

  • Envios gratis a todo el pais
  • Compra segura
  • Atencion rapida o WhatsApp

F. Seccion de video

No necesariamente espero que resuelvas la produccion del video completo, pero si quiero que contemples la pieza visual de entrada para esa seccion.

Referencia tecnica de Tienda Nube:

  • resolucion minima recomendada: `1280 x 720 px`
  • relacion de aspecto: `16:9`
  • formato horizontal

En Toluca esta seccion puede llevar:

  • imagen de espera o portada
  • titulo
  • subtitulo
  • descripcion
  • boton

G. Testimonios

Quiero contemplar al menos 3 testimonios iniciales.

Tamano visible en el editor actual: imagen de testimonio `250 x 250 px`.

Cada testimonio lleva:

  • imagen
  • nombre
  • descripcion o texto del testimonio

H. Pop-up promocional

No es mi prioridad absoluta, pero me gustaria dejarlo contemplado como opcional.

Tamano recomendado para Toluca: `375 x 190 px`.

8. Entregables resumidos

  • logo principal
  • isotipo o icono
  • favicon
  • 3 banners de carrusel en desktop
  • 3 banners de carrusel en mobile
  • 3 o 4 banners de categorias
  • 2 o 3 banners promocionales
  • 3 banners informativos con su estilo visual
  • propuesta de portada para video
  • 3 visuales de testimonios
  • opcional: 1 pop-up promocional

9. Tono visual esperado

Quiero que el sistema completo se sienta:

  • coherente
  • vendedor
  • moderno
  • agil
  • confiable
  • agradable

No quiero que cada seccion parezca de una marca distinta.

10. Lo que quiero evitar

  • piezas que se sientan copia de Mercado Top
  • una estetica demasiado marketplace generico
  • recursos infantiles o muy caricaturescos
  • banners sobrecargados
  • exceso de texto dentro de las piezas
  • visuales sin identidad
  • iconografia demasiado fria

11. Criterio para evaluar la primera iteracion

  • se siente realmente Mercado Box y no una copia?
  • transmite envio gratis, seguridad y rapidez?
  • la tienda se ve mas profesional y propia?
  • el icono tiene potencial de marca?
  • los banners funcionan comercialmente?
  • hay una identidad coherente entre todas las piezas?

12. Pedido final

Me gustaria que tomes como referencia la energia comercial de Mercado Top, pero usando tu creatividad para construir algo propio, original y con mas identidad para Mercado Box.

No busco solamente un logo lindo. Busco una marca y una home de e-commerce que se vean pensadas en serio.

Lo mas importante para mi hoy es que, al entrar a la tienda, se entienda enseguida esto:

  • Mercado Box es confiable
  • comprar es facil
  • el envio llega bien
  • hay tecnologia
  • hay beneficios reales
  • y la marca tiene personalidad propia
